# 92 Pizza Abbottabad 🍕

A fully responsive restaurant landing page built for 92 Pizza Abbottabad, showcasing their menu, special deals, and combo offers — designed to work smoothly on everything from a full-size laptop screen down to a small mobile phone.

🔗 Live Site: (add your GitHub Pages link here once deployed)


About This Project

This is a front-end website I built from scratch using plain HTML and CSS — no frameworks like Bootstrap, no external UI libraries. The goal was to design a clean, modern, food-ordering style layout (similar to platforms like Foodpanda) while keeping the code lightweight and fully hand-coded.

The site includes:


A sticky, blurred-glass style header with logo and quick action icons
A hero section with restaurant info, ratings, and address
A category slider/tab bar (Special Pizza / Pizza / Deal)
A Special Pizza grid section with menu cards
A Pizza section for regular items
A Deal section showcasing combo offers with pricing
A full footer with social links, quick links, locations, and contact/download info


Every section was made fully responsive across three breakpoints (tablet, mobile, and small mobile) so the layout never breaks or overflows on any screen size.


Tech Stack


HTML5 — semantic structure
CSS3 — Flexbox & CSS Grid for layout, no frameworks
Font Awesome — for icons (loaded via CDN)
